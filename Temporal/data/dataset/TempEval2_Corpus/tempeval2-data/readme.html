Tempeval-2 Data — Release Notes

Marc Verhagen - January 10th, 2011

This directory contains all Tempeval2 data previously released in the context of the SemEval-2010 workshop as well as the previously unpublished evaluation keys. More specifically, it includes:

  1. The second release of the training data from March 22nd, 2010;
  2. The first release of the test data from March 28th, 2010;
  3. Several minor fixes to the test data as distributed through the Google Tempeval group in April 2010;
  4. Scorer scripts that were distributed with the training and test data;
  5. An extra top-level folder with all documentation previously released as well as an added file for Spanish.
  6. The answer keys to the test data, not previously released.

The training data are in the training directory, the test data and subsequent fixes are in the test directory, and the scorer scripts are in the test directory. Answer keys are added to the test data for each language, in a directory named key.

It should be noted that for some languages the annotations are a bit experimental. For all languages but English, and to a lesser extent Italian, the Tempeval-2 annotation was the first temporal annotation of this kind.

For more information, check the release notes for the training data and the release notes for the test data.
